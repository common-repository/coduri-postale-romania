== Description ==

Zip codes for any location from Romania. This plugin calls a service from http://api.coduriro.ro to retrive data your users search for. You'll need to get an api key before installing the plugin. You can get it for free from http://www.coduriro.ro/#getApi. Enter your email address and the api key will be sent to your inbox instantly.

== Frequently Asked Questions ==

= What is the purpose of this plugin? =

This plugin will allow you to put on your website an widget that find any zip code from Romania or to find a location by searching for the zip code.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `coduri-postale` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget generated in any sidebar from your website.
4. Customise the style.css from plugin folder from Wordpress editor to change the look of the widget.
5. Go to http://www.coduriro.ro/#getApi to get a free api key.
6. Enter your api key in widget's settings (api key field).
7. Enjoy your new widget!
